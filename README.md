AI Operations Manager Stack

This repository contains the blueprints for an autonomous content-to-commerce machine.

Architecture

Input: Chapters stored in Google Drive.

Brain: content_engine.py processes text via LLM.

Execution:

HeyGen for Video.

Systeme.io for Funnel Lead Capture.

Make.com for Webhook orchestration.

Setup

Deploy main_orchestrator.json into Make.com.

Configure environment variables for OpenAI and Systeme.io.
